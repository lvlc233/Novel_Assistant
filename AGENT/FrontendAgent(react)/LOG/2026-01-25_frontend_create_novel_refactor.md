# 前端变更日志

## 基本信息
- **时间**: 2026年01月25日 14:10
- **目标**: 重构创建作品界面，实现分步向导（类型选择 -> 插件配置），优化用户体验。

## 变更详情
### 1. 组件重构 (`CreateNovelCard.tsx`)
- 将单页表单重构为两步向导：
  - **第一步**: 基础信息（封面、类型、标题、简介、关联知识库）。
  - **第二步**: 插件配置（左侧插件列表，右侧配置面板）。
- 新增作品类型选择 UI（网格布局）。
- 新增插件配置面板，支持模型选择和功能开关配置。
- 优化界面布局和交互动画。

### 2. 数据模型更新
- **`src/types/novel.ts`**: `Novel` 接口新增 `type` 和 `plugins` 字段。
- **`src/services/models.ts`**: DTO 新增对应字段。
- **`src/services/novelService.ts`**: 
  - 更新 `CreateNovelDto` 支持 `novel_type` 和 `plugins`。
  - 更新 Mock 实现以存储新字段。

### 3. 页面集成
- **`src/app/novels/page.tsx`**: 更新 `handleCreateNovel` 回调，将组件产生的数据正确传递给 Service。
