# 编辑器侧边栏交互优化日志

## 基本信息
- **时间**: 2026-01-21 13:05
- **目标**: 实现编辑器左右两侧面板（AI 助手、目录）的折叠/展开功能，并提供流畅的过渡动画。
- **变更范围**:
    - `src/components/editor/AIAssistant.tsx`: 新增折叠/展开逻辑，收起状态下显示悬浮气泡按钮。
    - `src/app/editor/page.tsx`: 管理 AI 助手的展开状态，实现左侧面板宽度的动态过渡（0px <-> 400px）。
    - `src/components/table-of-contents/TableOfContents.tsx`: 新增折叠状态 UI（竖排文字），支持通过 Header 按钮切换状态。
    - `src/components/editor/DocumentEditor.tsx`: 管理目录栏的折叠状态，实现宽度动态过渡（12px <-> 64px）。

## 验证方式与结果
1.  **验证方式**:
    - 访问编辑器页面。
    - 点击左侧 AI 助手顶部的“收起”按钮（ChevronLeft），面板应向左滑出消失，并在左下角出现悬浮气泡按钮。
    - 点击左下角悬浮气泡按钮，AI 助手面板应平滑展开。
    - 点击右侧目录栏顶部的“收起”按钮（ChevronLeft），目录栏应收缩为窄条，显示竖排文字。
    - 点击收缩后的目录栏，目录应平滑展开。
    - 快速切换两个面板的状态，中间的主编辑区应自适应宽度变化，无卡顿或布局错乱。

2.  **结果**:
    - 左右侧边栏均已实现可折叠交互。
    - 过渡动画流畅（`transition-all duration-300 ease-in-out`）。
    - 视觉效果符合预期，收起状态下的 UI 简洁美观。
