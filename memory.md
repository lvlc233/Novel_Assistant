## 🔧 技术栈总结

### 后端技术
- **框架**: FastAPI + SQLModel
- **数据库**: PostgreSQL + pgvector
- **AI集成**: LangGraph + LangChain
- **异步处理**: async/await

### 前端技术
- **框架**: Next.js 14 + TypeScript
- **状态管理**: Zustand
- **样式**: TailwindCSS + Radix UI
- **富文本**: Tiptap

### 架构模式
- **微内核插件化** - 核心最小化，功能插件化
- **BFF代理模式** - 统一数据代理接口
- **定义数据分离** - Schema定义与运行时数据分离
