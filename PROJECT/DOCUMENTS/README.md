# 介绍
Document里存放各自项目中生成的必须的或重要的文档信息,它通常应指导项目如何的进行推进,如何实现。推荐管理员将自己的文档上传到这里。

# 作者推荐
- 项目设计稿
- 某领域(如Agent)设计稿
- 流程编排稿
- 系统构架图
- 项目环境说明
- 敏感信息配置
- ...
